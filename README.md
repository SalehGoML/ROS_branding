# ROS — Intelligent Brand Management Platform

A modern, full-stack platform for strategic brand consulting and AI-powered brand analysis, built for the Persian-speaking market.

---

## Tech Stack

**Frontend** — Next.js 16, TypeScript, Tailwind CSS  
**Backend** — Go (Gin)  
**Database** — PostgreSQL  
**Deployment** — Go serves Next.js static export

---

## Overview

ROS is a brand agency platform that combines human strategic thinking with data-driven analysis. It provides a suite of tools for brand strategy, visual identity, and intelligent brand monitoring across all touchpoints.

**Core capabilities:**
- Brand strategy consulting and roadmap design
- Visual identity and naming systems
- AI-powered brand coherence analysis across channels
- Client dashboard with real-time brand health metrics

---

## Getting Started

```bash
# Install dependencies
cd frontend && npm install

# Run development server
npm run dev        # → http://localhost:3000

# Production build
npm run build      # → outputs to /out
```
